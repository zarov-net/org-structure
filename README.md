# API организационной структуры

Сервис для управления иерархией подразделений и сотрудниками компании.

## Возможности

- Создание, просмотр, редактирование и удаление подразделений
- Древовидная структура подразделений с настраиваемой глубиной
- Добавление сотрудников в подразделения
- Два режима удаления: каскадный и с переназначением сотрудников
- Валидация входных данных и проверка бизнес-правил
- Документация API через Swagger UI
- Логирование запросов и восстановление после паник

## Технологии

- Go 1.21+
- PostgreSQL 16
- GORM (ORM)
- goose (миграции)
- net/http (стандартный роутер)
- Swagger (документация)
- Docker, docker-compose
